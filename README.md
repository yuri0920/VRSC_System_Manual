# VR-SCとは
- VR空間上に相談者役と相談相手役の2つのアバターを配置し、利用者が両アバターの視点を交互に入れ替わりながら自分の悩みについて自己対話する手法。利用者はまず相談者役アバターに没入し、自分の悩みを話す。話し終えたら相談相手役アバターに支店が移動し、先ほど自分が話した内容の録音が再生される。相談相手の立場になったつもりで自分の悩みを聞き、それに対してアドバイスや質問などをする。その後利用者は再び相談者役のアバターに視点を戻し、相談相手役アバターの視点で行った発言の録音を聞く。録音を聞き終えたら、それに対して相談者の立場から応答し、話し終えたら再び相談相手役アバターへ視点を移動させる。以上のプロセスを繰り返し、自分の悩みについて一人二役で話し合う。
