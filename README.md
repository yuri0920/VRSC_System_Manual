# VRセルフカウンセリング（VR-SC）とは
- VR空間上に相談者役と相談相手役の2つのアバターを配置し、利用者が両アバターの視点を交互に入れ替わりながら自分の悩みについて自己対話する手法です。
  - 利用者はまず相談者役アバターに没入し、自分の悩みを話す。
  - 話し終えたら相談相手役アバターに支店が移動し、先ほど自分が話した内容の録音が再生される。相談相手の立場になったつもりで自分の悩みを聞き、それに対してアドバイスや質問などをする。
  - その後利用者は再び相談者役のアバターに視点を戻し、相談相手役アバターの視点で行った発言の録音を聞く。録音を聞き終えたら、それに対して相談者の立場から応答し、話し終えたら再び相談相手役アバターへ視点を移動させる。
- 以上のプロセスを繰り返し、自分の悩みについて一人二役で話し合うものになります。
# アーカイブの内容
- アーカイブには以下のフォルダが含まれます。
  - 実験システム：実際に使用したのプロト版システムと各参加者が実際に使用したシステムがあります。
  - 測定項目・分析データ：実験に使用した説明文書や質問紙、収集したデータとその分析用コードを入れてます。
## 実験システム
- 実験条件（セルフ条件・フロイト条件・友人条件）ごとのシステムのプロト版と、そこから実験において各参加者が実際に使用したシステム（参加者の顔写真を元に作成したアバターを使用したもの）があります。
- システム１つにつき容量が大きいため。個別に圧縮・アップロードしています。必要なバージョンのシステムを都度ダウンロード・展開して使用してください。

## 測定項目・分析データ
- 「説明文書・質問項目」、「分析」、「対話ログ」の３つのフォルダがあります。
- 「説明文書・質問項目」、「分析」はzip形式で圧縮しておりますので、ダウンロード・展開してご使用ください。
- 「対話ログ」は7zip形式で圧縮しております。解凍方法は<a href="7zipの解凍.md">7zipの解凍</a>を確認してください。
### 説明文書・質問項目
- 実験で参加者に渡した募集要項、説明文書、同意書など
- 実際に測定した項目内容
  - 健康アンケート
### 分析
- 「20260115分析用データセット.xlsx」
  - 参加者の<b>同意書への署名</b>（「1日目_row」のシート内）
  - 質問紙への回答のRAWデータ
  - Rにて記述統計量の算出、ART分散分析を行うためのデータセット
- 「20260118個別分析.xlsx」
  - アバターに対する身体化感覚・当人との類似度の評価が高かった参加者を抽出し、条件間での傾向の差異を比較した結果
- 「2026115ART分散分析.R」
  - 「20260115分析用データセット.xlsx」を用いてRでデータ整形、記述統計量の算出、箱ひげ図の描画、ART分散分析を行うためのコード
- 参加者へのインタビュー
  - 各参加者にVR-SC終了後に口頭で行ったインタビューのメモ
  - boxのアーカイブには一部の参加者のもののみアップロード。その他の参加者のメモは紙媒体（必要な場合は菊池先生にご確認ください）。
### 対話ログ（音声・文字起こし）
- VR-SC中の参加者の対話内容の録音とその文字起こしになります。
- 「dialog」フォルダ
  - この中に参加者のIDごとに相談者役アバター視点と相談相手役アバター視点での参加者のそれぞれの発話内容の録音と、それを文字起こしした「相談者.xlsx」・「相談相手.xlsx」、対話形式にまとめなおした「対話.xlsx」を入れています。
  - 録音音声は各参加者のVR-SCシステムのプロジェクトのフォルダ内の「Dialog_Player1」、「Dialog_Player2」内にも同じものがあります。
- 「Transcription_20251229.py」
  - 上記の文字起こしの作成に使用したPythonのコードです。
  - Wishperを使用して自動で文字起こししているため、誤字や抜けがあるかもしれません。
