# 音声変換用プログラム（Python）の起動
- 以下の作業はフロイト条件、友人条件のシステムのみ必要になります。セルフ条件のシステムの場合はこのページを飛ばして次の<a href="VR-SCシステムの起動・使用方法.md">VR-SCシステムの起動・使用方法</a>へ進んでください。
## 事前準備
- 「VoiceChange_〇〇_〇〇.py」を開く
  - プロト版：boxの「k-lab/docs/2025archive/2025中村勇理_アーカイブ/実験システム/Proto」の中に「VoiceChange.py」
  - 参加者用：解凍した各参加者のフォルダ「{ID}/Voice」の中に「VoiceChange_〇〇_〇〇.py」
- 以下の変数を指定
  - input_folder_path：UE5のProjectフォルダの中の「Dialog_Player2」（システムが録音した音声が保存されるフォルダ）のパスを指定
  - output_folder_path：UE5のProjectフォルダの中の「Dialog_Player2_Change」（変換後の音声が保存されるフォルダ）のパスを指定
  - model_path：RVCで学習したモデルのパスを指定
  - pitch：変換前の音声ーモデル音声間の半音差を入力
  - device：変換の際に使用する計算資源を指定（CPUを使用する場合は"cpu"、GPUを使用する場合は"cuda:0"）

- プログラムを保存
## 起動方法
- コマンドプロンプトを起動
- pythonファイルがあるディレクトリまで移動
  - 音声変換用のファイル（「VoiceChange_〇〇_〇〇.py」）が入っているフォルダを左クリック→「パスのコピー」
  - コマンドプロンプト上でディレクトリを移動
    ```
    cd {コピーしたパス}
    ```
    <br><img width="503" height="31" alt="スクリーンショット 2026-03-24 154508" src="https://github.com/user-attachments/assets/a0951251-b59d-409e-bb87-3be8877cdd6c" />

- pythonファイルの実行
  <br>（"{変換前の音声ファイルが保存されるフォルダのパス}" 音声ファイル追加を監視中... " と表示されたらOK）
  ```
  python　{pythonファイル名}
  ```
